# Crawford County JFS Website Content Archive

This directory contains a complete scrape of the Crawford County Job and Family Services website (https://crawfordcountyjfs.org/).

## Structure

The content is organized to mirror the website's structure, with each major section in its own folder:

- **home/** - Homepage content
- **ohio-means-jobs/** - Employment and job services
- **child-protective-services/** - Child and adult protective services
  - **adult-protective-services/** - Elder abuse reporting
  - **kinship/** - Kinship care programs
- **foster-care-adopt/** - Foster care and adoption services
- **public-assistance/** - SNAP, Medicaid, and cash assistance
- **child-support/** - Child support enforcement
- **additional-resources/** - Community resources and partners
- **join-our-team/** - Employment opportunities
- **current-rfps/** - Requests for proposals
- **report-child-abuse/** - Child abuse reporting information
- **fraud-assistance/** - Fraud reporting
- **privacy-policy/** - Terms and privacy policy

## Content Format

All content has been saved in Markdown (.md) format for easy reading and portability. Each section contains an `index.md` file with the full text content from that page.
